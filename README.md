# AIML Python Learning Repository

Structured Jupyter notebooks for Python fundamentals through advanced topics, organized by learning modules.

## Modules

| Module | Folder | Topics |
|--------|--------|--------|
| **Module 2** | [Module2/](Module2/) | OOP, Exception Handling, File Handling, Decorators, Generators, Context Managers |
| **Module 3** | [Module3/](Module3/) | Multithreading, Multiprocessing, AsyncIO, Collections, Itertools, Logging, Dataclasses, Typing, Performance |
| Basics | [BasicPython/](BasicPython/) | Variables, control flow, data structures, and introductory notebooks |
| Advanced | [AdvancedPython/](AdvancedPython/) | Supplementary advanced examples and package demos |

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate          # Windows
# source .venv/bin/activate     # Linux/macOS
pip install -r requirements.txt
```

Open any `.ipynb` file in Jupyter Lab, VS Code, or Cursor and run cells top to bottom.
